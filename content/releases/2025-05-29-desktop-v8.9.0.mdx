## Shinny new things

- **Table view**: Add a table view to your task management. [docs](/docs/task-view/table-view)

## Features

- **Tasks view in base**: Support both file tasks and also normal inline tasks in Base plugin of Obsidian.

## Fixes

- **Auto complete parent**: Sometimes line with indent will not work correctly when [auto complete parent](/docs/task-status#parent-task-automation) is enabled.
- **Conflicts**: Fixed conflicts when [date append](/docs/append-date) and [reward](/docs/reward) are enabled at the same time.
- **Append date**: Fixed issue when move line up or down will cause [date append](/docs/append-date) not working correctly.
- **Sidebar of task view**: Order of default sidebar items should be on the top of the list.
