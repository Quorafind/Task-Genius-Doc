Project view is a view that allows you to view your tasks by their project. Inherits from the [Two Column View](/docs/task-view/two-column-view).

<VersionCallout version="7.0.0" />

## Open the project view

1. Open the [Task Genius View](/docs/task-view/index).
2. Select `Project` nav item in the left sidebar.
3. You will see the tasks grouped by project in the right panel.

## Usage

1. Add `#project/xx` to your tasks you want to group by project.
2. Open the [Task Genius View](/docs/task-view/index).
3. Select `Project` nav item in the left sidebar.
4. You will see the tasks grouped by project in the right panel.

### Multi-select

Read more about the multi-select in the [Two Column View](/docs/task-view/two-column-view#multi-select) page.

## Settings

You can read more about the settings in the [Basic view settings](/docs/task-view/basic-view-settings#basic-settings) page.
