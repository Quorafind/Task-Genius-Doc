## Usage Examples

### Basic Usage (Primary by default)
```mdx
<DeepLinkButton href="obsidian://task-genius/settings">
  Open Settings
</DeepLinkButton>
```

### With Variant
```mdx
<DeepLinkButton href="obsidian://task-genius/settings" variant="secondary">
  Secondary Action
</DeepLinkButton>
```

### Custom Fallback Text
```mdx
<DeepLinkFallback customText="Your custom fallback message here" />
```
